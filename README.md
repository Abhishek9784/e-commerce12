## Run Locally

Clone the project

```bash
 https://github.com/Abhishek9784/e-commerce12.git
```

Go to the project directory

```bash
  cd e-commerce12
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
